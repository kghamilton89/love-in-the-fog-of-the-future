# Любовь в Тумане Будущего

## Overview

[Любовь в Тумане Будущего: История Одного Романа в 4560 Году](https://ru.wikipedia.org/wiki/%D0%9B%D1%8E%D0%B1%D0%BE%D0%B2%D1%8C_%D0%B2_%D1%82%D1%83%D0%BC%D0%B0%D0%BD%D0%B5_%D0%B1%D1%83%D0%B4%D1%83%D1%89%D0%B5%D0%B3%D0%BE) ([Love in the Fog of the Future: The Story of a Romance in the Year 4560](https://en.wikipedia.org/wiki/Love_in_the_Fog_of_the_Future)) is the only known novel by Russian author Andrei Marsov. This repository contains contains the original Russian text transcribed directly from a copy of the novel in the [Russian National Library](http://nlr.ru/) in Saint Petersburg, Russia as well as an English translation.
